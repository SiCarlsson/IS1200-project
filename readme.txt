To compile the code: 
    - Open a mcb32 instance in your terminal
    - Navigate to the game folder (IS1200-project)
    - Make sure your UNO32 Board is connected to your computer
    - In your terminal, give the command 'make'

To install and run the code on a ChipKIT UNO32 Board:
    - Make sure your UNO32 Board is connected to your computer
    - In your terminal, give the command 'make install'

Disclaimer:
    - The make file has a specified USB-port, please update it to match your computer
